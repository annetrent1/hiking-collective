# Hiking Collective

Welcome to the Hiking Collective! This is the place where hikers can find the area they live in or are visiting and join a hiking group.

## Getting Started

```bash
npm install
```
Go to server and start
```bash
cd server/
npm run start
```
Go to client and start
```bash
cd client/
npm run start
```
